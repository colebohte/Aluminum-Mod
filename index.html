How to Install Aluminum



1. Click "Win+R" Keys on your keyboard

2. Type or Copy "C:\Program Files (x86)\PhoenixCoder\Mini Mayhem\resources\app\assets" into the text box, Click OK

3. Paste the Aluminum assets into the folder

4. If prompted, Click Replace files

5. Open Mini Mayhem!
